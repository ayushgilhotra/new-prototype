WipeSure Desktop Application - Installation Instructions
=========================================================

IMPORTANT: This desktop application is designed to run on your local computer, 
not in the cloud environment. It will actually delete files from your device.

Installation Steps:
1. Copy all files from the desktop-package folder to your local computer
2. Make sure you have Node.js installed on your computer
3. Open terminal/command prompt in the copied folder
4. Run: npm install
5. Run: npm start

The application will open and connect to your WipeSure server for processing
while having the ability to delete local files from your device.

WARNING: This application will permanently delete files from your device
after generating certificates. Use with caution.
